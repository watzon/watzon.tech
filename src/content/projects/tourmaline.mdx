# Tourmaline

Tourmaline is a comprehensive Crystal framework for Telegram bot development that provides a clean, type-safe API for building sophisticated bots and automation tools.

## Getting Started

```crystal
require "tourmaline"

class EchoBot < Tourmaline::Client
  @[Command("echo")]
  def echo_command(context, message)
    text = message.text.not_nil![5..-1] # Remove "/echo "
    context.bot.send_message(chat_id: message.chat.id, text: text)
  end
end

bot = EchoBot.new(ENV["TELEGRAM_TOKEN"])
bot.poll
```
